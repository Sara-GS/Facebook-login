# Facebook OAUTH login

Ejercicio donde se nos pide desde el backend crear un sistema de autenticación de usuarios a través de la red social Facebook.
En la Welcome page del proyecto, he añadido enlace al repositorio de github y a la página de Nuwe de donde se ha sacado este reto.

## Requisitos del proyecto

Los social login son una opción cada vez más popular y es importante saber implementarlos y navegar entre la documentación que proponen las diferentes empresas.

Para ayudar a mejorar la interacción con los usuarios este proyecto se basa en crear un pequeño server configurado para crear usuarios y permitir hacer login utilizando el social login de Facebook.

Los objetivos y tareas que hay que cumplir se pueden observar en la sección User Stories / Objetivos.

### Objetivos (User Stories)

✅ Task 1 → Crear un server básico que en la primera ruta diga: "Facebook social login implementation"

✅ Task 2 → Implementar un modelo de usuario básico dónde guardarlos la key que nos devuelve el social login

✅ Task 3 → Implementar un endpoint que nos permita hacer social login utilizando la estrategia de Facebook

✅ Task 4 → Configurar la base de datos dónde poder guardar los datos del usuario

✅ Task 5 → Crear una middleware de protección de rutas que no te deje pasar si no has iniciado sesión.

#### Stack

Proyecto creado con PHP, Laravel 9.

##### Instalacion

npm install
npm run dev
composer install
composer require laravel/ui
